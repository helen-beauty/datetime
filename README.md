This is a command line tool which calculates sunrise, sunset and day length for one specific location - my hometown.

One sole reason to create this tool, and not use common websites with same information is the function of seeking the date within next 365 days of date with same day length as current. I call it mirroring day.

The tool written in rust as #I'm practicing in this language. And this is my first project where I used AI only for couple functions which I didn't wish to invent. I may find them below in source, they had appropriate comment.
